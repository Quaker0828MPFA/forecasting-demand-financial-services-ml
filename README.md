# 📊 Forecasting de Demanda con Machine Learning

Proyecto de ciencia de datos enfocado en la predicción de demanda de solicitudes financieras y optimización de capacidad operativa.

## 🔍 Problema
Predecir la demanda diaria de solicitudes y calcular la cantidad óptima de recursos humanos necesarios.

## ⚙️ Tecnologías
- Python
- Scikit-learn
- SQL
- Power BI

## 📈 Modelos utilizados
- ARIMA
- Holt-Winters
- Random Forest (Forecaster Autoregresivo)
- LSTM

## 🏆 Resultados
- Mejor modelo: Random Forest autoregresivo
- MSE: 0.224
- Reducción de incertidumbre en planeación operativa

## 🚀 Impacto
Optimización de recursos y mejora en la toma de decisiones basada en datos.
